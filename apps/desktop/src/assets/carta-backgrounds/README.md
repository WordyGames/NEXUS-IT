# Imágenes de Fondo para Cartas Responsivas

Coloca las siguientes imágenes en esta carpeta para que aparezcan como marca de agua en las cartas responsivas:

## Archivos necesarios

1. **amex.png** → GRUPO AMEX
2. **aromata.png** → ESPECIAS NATURALES DEL NORTE  
3. **liumaq.png** → EQUIPOS OSENAL

## Recomendaciones

- **Formato:** PNG (preferido) o JPG
- **Opacidad:** Recomendado 10-20% de opacidad para no opacar el texto
- **Resolución:** Mínimo 1200px de alto
- **Posición:** La imagen se centrará automáticamente en el PDF

## Cómo preparar las imágenes

Si tus imágenes originales son muy opacas:

1. Abre la imagen en un editor (Photoshop, GIMP, Paint.NET)
2. Reduce la opacidad a 10-20%
3. Guarda como PNG con el nombre exacto

## Nombres de archivo EXACTOS

```text
amex.png
aromata.png
liumaq.png
```

**Importante:** Los nombres deben ser exactamente como se muestran arriba (minúsculas, sin espacios).

---

Una vez colocadas las imágenes aquí, las cartas responsivas las incluirán automáticamente como fondo.
